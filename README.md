## 📖 Overview
CancerAI Health Assistant is a Flask-based intelligent healthcare platform that enables users to analyze symptoms, upload medical reports for OCR extraction, obtain AI-assisted insights, manage their profile, chat with an AI assistant, and discover nearby hospitals through a clean responsive interface.

## ✨ Core Features
- 🔐 Email & Google OAuth authentication
- 👤 User profile management
- 📄 Medical report upload
- 🔎 OCR text extraction (OCR.Space API)
- 🤖 AI symptom analysis
- 🩺 Cancer risk prediction
- 🏥 Hospital recommendation
- 💬 AI chatbot
- 📊 Dashboard & analytics
- 📱 Responsive UI

## 🚀 Installation
```bash
git clone https://github.com/<your-username>/CancerAI.git
cd CancerAI
python -m venv venv
# Windows
venv\Scripts\activate

```

## 🔄 Workflow
1. Login/Register
2. Upload report or enter symptoms
3. OCR extracts report text
4. ML predicts risk
5. Chatbot & hospital recommendations
6. Dashboard visualizes results
